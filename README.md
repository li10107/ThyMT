This repository contains the source code for the paper **"ThyMT: A Unified Multitask Learning Framework for Thyroid Nodule Diagnosis via Ultrasound Imaging"**.
<img width="1791" height="1224" alt="image" src="https://github.com/user-attachments/assets/c5256821-7a05-45b0-9741-faf634fdd2fb" />


### Code Structure
*   `USUnimodelSingletaskModel.py`: Implements the single-task model.
*   `UnimodalMultitaskModel.py`: Implements the multitask model.

### Dataset Availability
